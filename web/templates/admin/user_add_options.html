User Addition Recommendations
Voice Sample Requirements:
  • Minimum 5 WAV files per user
  • Duration of each recording: 3-10 seconds
  • Format: WAV, 16kHz, 16bit, mono
  • Good sound quality, minimal background noise
For Batch Addition:

Prepare file structure:

directory/
├── John_Smith/
│   ├── recording1.wav
│   ├── recording2.wav
│   └── ... (at least 5 recordings)
├── Jane_Doe/
│   ├── recording1.wav
│   └── ...
└── ...
