Directory Preparation Instructions

To import users from a directory, prepare your files as follows:

directory/ ├── John_Smith/ │ ├── recording1.wav │ ├── recording2.wav │ └── ... (at least 5 recordings) ├── Jane_Doe/ │ ├── recording1.wav │ └── ... └── ...

Each subdirectory will be treated as a separate user. The directory name will be used as the user's name.

Important: In Windows, you cannot directly upload an entire directory through the browser. Choose one of these options:

  1. Package the directory into a ZIP archive and use the "Import from ZIP" tab
  2. Select all subdirectories and files manually (holding Ctrl)

ZIP Archive Preparation Instructions

To import users from a ZIP archive, prepare your archive with the following structure:

archive.zip ├── John_Smith/ │ ├── recording1.wav │ ├── recording2.wav │ └── ... (at least 5 recordings) ├── Jane_Doe/ │ ├── recording1.wav │ └── ... └── ...

Each subdirectory in the root of the archive will be treated as a separate user.
