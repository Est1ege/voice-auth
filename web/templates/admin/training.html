Тренировать модель

Рекомендации по тренировке
  • Модель распознавания голоса следует тренировать, когда добавляются новые пользователи. Чем больше образцов голоса, тем лучше будут результаты распознавания.
  • Модель защиты от спуфинга следует периодически переобучать с добавлением новых типов поддельных голосов для повышения защиты.
  • Тренировка выполняется на доступном оборудовании. Если доступен GPU, процесс будет значительно быстрее.
  • Текущее оборудование: {{ 'GPU' if 'cuda' in device else 'CPU' }}
Тренировка моделей выполняется в фоновом режиме и может занять продолжительное время. Вы можете продолжать работу с системой во время тренировки.
